# HandiMap


## Overview
Our team BBY17 is developing a web app named HandiMap to help students with accessibility needs at BCIT to navigate accessible paths around campus with an interactive map and open-sourced reviews.

---


## Styles

- Look at a list of curated hiking trails with images and details
- Mark and unmark trails as favorites
- View a personalized list of favorite hikes
- Responsive design for desktop and mobile

---


## Technologies Used

- **Frontend**: HTML, CSS, JavaScript
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Backend**: Firebase for hosting
- **Database**: Firestore
- **MapBox**: Mapbox
---


## Usage

1. Open your browser and visit `http://localhost:5137`.
2. Browse the map and choose which rooms you have classes in.
3. Click the question mark icon to view that room's reviews to check for accessibility.
4. View and update your profile in the profile section.
5. Toggle the map markers and sign out in the settings section.
